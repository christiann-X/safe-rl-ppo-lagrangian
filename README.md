## 项目描述

**核心痛点**：传统强化学习仅追求累积奖励最大化，在自动驾驶等安全攸关场景中，智能体可能为了更快到达目标而频繁驶入危险区域，产生不可接受的安全风险。本项目通过 PPO-Lagrangian 算法引入安全约束机制，使智能体在"高效导航"与"主动避障"之间取得可量化的平衡。

**核心逻辑流**：采用**双 Critic 架构 + 拉格朗日对偶**的端到端学习范式，不含多 Agent 协作。推理链路为：Actor 输出动作 → 环境返回 reward 和 cost → Reward Critic 和 Cost Critic 分别估计两个维度的状态价值 → GAE 计算两路独立优势函数 → 拉格朗日乘子 λ 将两路优势加权合并为联合优势 $A_{combined} = A_{reward} - \lambda \cdot A_{cost}$ → PPO 裁剪目标更新策略。λ 自身通过 SGD + EMA 代价平滑 + L2 正则化进行自适应调节：代价超标时 λ 增大以加强安全惩罚，代价达标时 λ 衰减以放宽约束，形成**单 Agent 内的奖励-安全博弈闭环**，无需外部监督或多方协商。

**训练策略含长链分阶段推理**：Phase 1（极低 λ 学习率先学导航）→ Phase 2（逐步引入约束感知）→ Phase 3（加速 λ 增长试探有效区间）→ Phase 4（λ 大幅进入 1.0-3.0 约束区间），通过分析 λ=2.0 附近 Reward≈30、Cost≈44 的最佳平衡点，验证了适度安全约束优于零约束和过强约束的结论。
